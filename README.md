
# AI Chess Game (Python)

This project is a chess game implemented in Python featuring an AI opponent using the Minimax algorithm with Alpha-Beta pruning.

## Features
- Play against AI
- Minimax decision-making
- Alpha-Beta pruning optimization
- Object-oriented design
- Simple text or GUI interface

## Technologies
- Python
- Algorithms & Data Structures
